# Random-Password-Generator-App
#Overview
The Random Password Generator is a Python-based application that allows users to create secure, random passwords. This tool offers users the flexibility to customize their passwords by choosing the number of uppercase letters, lowercase letters, digits, and special characters. With its simple and intuitive graphical user interface (GUI) built using PySimpleGUI, this application helps users easily generate strong, unpredictable passwords for various purposes.

#Features
Customizable Password Generation: Choose the number of uppercase, lowercase, numeric, and special characters.
Randomization: The order of characters is completely randomized for increased security.
User-friendly GUI: Built using PySimpleGUI, the app has an easy-to-use interface.
Error Handling: Provides feedback if invalid input is given, ensuring smooth operation.

#Usage
Open the application.
Input the number of uppercase, lowercase letters, digits, and symbols you want in your password.
Click OK to generate your password.
The generated password will be shown on the screen for you to copy and use.
Press Cancel to close the application or generate a new password.

#Purpose of the Random Password Generator
The purpose of the Random Password Generator is to:

Generate Secure Passwords: The primary goal is to help users create strong, unpredictable passwords that are difficult to guess or crack. This ensures better security for online accounts, databases, or systems.

Customization: It allows users to define the number of uppercase letters, lowercase letters, digits, and special characters. This provides flexibility to users who need to meet specific password requirements (e.g., for websites or corporate systems).

Simplify the Process: With its easy-to-use GUI, the app simplifies the process of generating random passwords without requiring users to know programming or complex tools.
